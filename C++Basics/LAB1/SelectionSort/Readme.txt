Анализ времени выполнения алгоритма для разных входных данных:

Для алгоритма сортировки выбором, его временная сложность составляет O(n^2). Это означает, что время выполнения алгоритма увеличивается квадратично с увеличением 
количества элементов во входных данных.

Для небольших наборов данных алгоритм работает достаточно быстро. Однако, с увеличением размера входных данных, время выполнения алгоритма значительно увеличивается. 
Например, для списка из 100 элементов алгоритм сортировки выбором будет работать значительно дольше, чем для списка из 10 элементов.
