Алгоритм сортировки пузырьком - это простой алгоритм сортировки, который многократно проходит по списку, сравнивая соседние элементы и меняя их местами, 
если они стоят в неправильном порядке. Алгоритм продолжает выполнение до тех пор, пока не будет произведен ни один обмен.

Основные шаги алгоритма сортировки пузырьком:
1. Проходим по списку и сравниваем каждую пару соседних элементов.
2. Если элементы стоят в неправильном порядке, меняем их местами.
3. Продолжаем проход по списку, сравнивая и меняя элементы местами до тех пор, пока не будет произведен ни один обмен.
4. Результирующий список отсортирован.

Отчет об алгоритме сортировки пузырьком:
Алгоритм сортировки пузырьком - это простой алгоритм, который легко понять и реализовать. Он имеет временную сложность O(n^2), 
что делает его неэффективным для больших списков элементов. Однако, алгоритм подходит для небольших списков или в случаях, когда список уже почти отсортирован.

Принцип работы алгоритма заключается в многократном проходе по списку, сравнении соседних элементов и их перестановке, если они стоят в неправильном порядке. 
Хотя алгоритм неэффективен для больших списков, его простота делает его полезным инструментом при обучении основам сортировки и алгоритмов в целом.
