- Поле numerator представляет числитель дроби.
- Поле denominator - представляет знаменатель дроби.
- Свойство Numerator предоставляет доступ к числителю, а свойство Denominator - к знаменателю. Они также обеспечивают упрощение дроби при изменении числителя или знаменателя.
- Свойство DecimalValue возвращает десятичное представление дроби только для чтения.
- Конструктор Fraction принимает числитель и знаменатель дроби.
- Метод Simplify упрощает дробь, используя наибольший общий делитель.
- Метод GCD находит наибольший общий делитель двух чисел.
- Перегруженные операторы +, -, *, / позволяют выполнять операции сложения, вычитания, умножения и деления для объектов типа Fraction.
